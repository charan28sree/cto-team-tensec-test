Security is often about boring consistency, not perfection. Here are five high-impact steps that reduce risk quickly.

1) Turn on MFA (prefer app/passkey over SMS)

  • Enable MFA on your email account first — it’s the reset key for everything else.
  • If available, use passkeys or an authenticator app.
  • Avoid SMS when you have better options (SIM swaps are real).

2) Use a password manager (and rotate the worst offenders)

  • Pick one reputable manager and commit to it.
  • Start by changing passwords for: email, banking, cloud storage, and your primary social accounts.
  • Use unique, randomly generated passwords everywhere.

3) Patch aggressively (especially browsers + OS)

  • Enable automatic updates for your OS and browser.
  • Remove unused apps/extensions — fewer moving parts, fewer vulnerabilities.

4) Verify backups (a backup you haven’t tested doesn’t exist)

  • Keep at least one offline or “immutable” copy if possible.
  • Test a restore of a single file today (10 minutes now can save days later).

5) Add a phishing speed bump

  • Before clicking a link, ask: "What is this trying to get me to do?"
  • For important accounts, navigate by typing the site manually or using a bookmark.
  • If a message creates urgency, assume it’s a trap until verified.

Bonus: device encryption + screen lock

If your laptop/phone is lost or stolen, encryption + a strong lock screen prevents “physical access = game over.” Enable full-disk encryption and set a non-trivial PIN/passcode.
