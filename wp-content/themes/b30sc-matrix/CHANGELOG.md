# Changelog

All notable changes to the B30SC Matrix theme will be documented in this file.

## [1.0.0] - 2024-01-01

### Added
- Initial theme release
- Full WordPress block theme (FSE - Full Site Editing) support
- Matrix-inspired black/green cybersecurity aesthetic
- Comprehensive color palette (10 colors + 4 gradients)
- Custom typography with OCR A and Courier New fonts
- Theme.json configuration for global colors, fonts, and spacing
- Main stylesheet (style.css) with extensive CSS
- Editor-specific styles (style-editor.css)
- Functions.php with theme setup hooks

### Templates
- Front page template (front-page.html) with hero, services, and CTA
- Blog archive template (archive.html) with query loop
- Single post template (single.html) with related posts
- Index template (index.html) for generic listings
- Page template (page.html) for custom pages
- 404 error page (404.html)
- Search results template (search.html)

### Template Parts
- Header with site logo, title, and navigation
- Footer with three-column layout and copyright info

### Block Patterns
- Hero section pattern - Full-width hero with headline and CTAs
- Services section pattern - Three-column service showcase
- Features section pattern - Four-column feature list
- Call-to-action pattern - Prominent CTA with split layout

### Documentation
- Comprehensive README.md with installation and customization guides
- Screenshot.txt visual reference
- CHANGELOG.md (this file)
- Language files for translations (POT file)

### CSS Features
- Neon glow effects on headings
- CRT scanline animation
- Green accent borders on elements
- Hover effects with scale and glow
- Responsive design with mobile breakpoints
- Print-friendly styles
- Matrix-themed visual effects throughout

### Browser Support
- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
- Mobile browsers (iOS Safari, Chrome Mobile)

### Code Quality
- Valid theme.json with WordPress 6.0+ standards
- Proper file structure following WordPress conventions
- PHP functions with proper naming and documentation
- HTML templates using WordPress block markup
- CSS organized by component and functionality
- Full support for block editor customization
